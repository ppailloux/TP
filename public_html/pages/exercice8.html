LES OBJETS JSON

Créer des objets JSON dont les caractéristiques sont celles des objets du projet Sucrerie.
A l'instar de l'exercice précédent, faites des fiches en HTML pour chaque objet à afficher.

Pour cela, un constructeur par objet sera nécessaire mais une seule méthode pour afficher la fiche est demandée.
Elle devra donc prendre en paramètre l'objet et éventuellement un paramètre permettant de savoir dans quelle fiche placer l'objet.
